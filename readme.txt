サービスとは
アクティビティとは別に、裏で動いているプログラムのこと。
１枚目の画面で音楽を鳴らしていて画面が切り替わる時に、
アクティビティの中で音楽の呼び出しをしていれば、
画面が切り替わった時に音楽が止まってしまう。
画面が切り替わっても音楽を流しつつけたいときに使うのがサービス。
サービスには２種類あり、startServiceとbindServiceがある。
startServiceは、呼び出されてとまれと言われないと止まらず。
bindServiceは、呼び出した側のつながりが切れると止まる。

例えると、やる気のあるバイトは社員がいなくても頑張るが、
やる気のないバイトは社員がいる前ででしか頑張らないようなもの。
